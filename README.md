# TT9 Smart

A modernized T9 keyboard for Android phones with a physical 9-key keypad.

Fork of [Traditional T9](https://github.com/sspanak/tt9) by sspanak.

> **Status: early beta** — tested on Qin F21 Pro+ (Android 11). Other devices may work but are untested.

---

## What's different from Traditional T9

- **Gboard-style suggestion bar** — word suggestions appear inline as you type, tap to accept
- **Auto-accept** — configurable timeout to automatically accept the top suggestion
- **Case-aware suggestions** — typing `BONJ` suggests `BONJOUR`, typing `Bonj` suggests `Bonjour`
- **Voice input** — works with Futo Voice Input and other alternative recognizers, without stealing focus from your text field
- **Configurable hotkeys** — remap any key to any action
- Cleaner UI — removed Classic and Numpad layouts, reduced settings clutter

---

## Requirements

- Android 8.0+
- A physical 9-key keypad (also works on touchscreen)

---

## Install

Download the latest APK from [Releases](../../releases/latest) and install it manually.

> Enable "Install from unknown sources" in your Android settings if prompted.

---

## Languages

40+ languages supported with predictive text dictionaries. Load them from **Settings → Languages**.

## Privacy

No data collection, no telemetry, no network connections except to download dictionaries. See [privacy policy](docs/privacy.md).
